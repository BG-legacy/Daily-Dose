## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/yourusername/daily-dose.git
   ```

2. **Jump into the project**
   ```bash
   cd daily-dose
   ```

3. **Install the magic**
   ```bash
   npm install
   ```

4. **Set up your environment**
   - Create your `.env` file with your API keys
   - Configure your AWS and OpenAI credentials

5. **Launch the app**
   ```bash
   npm run dev
   ```
